# Contributing

Review the following guidelines for submitting issues or changes to this repository.

## Issues

If you encounter an issue with this project, you're welcome to submit a [bug report](https://github.com/IBM/secrets-manager-management-node-sdk/issues) to help us improve. Before you create a new issue, please search for similar issues in case someone has already reported the same problem.

## Changes

Thank you for contributing to IBM Cloud SDK projects! To learn more about submitting changes to this repository, check out the [IBM Cloud SDK Common contribution guidelines](https://github.com/IBM/ibm-cloud-sdk-common/blob/master/CONTRIBUTING_nodejs.md).
